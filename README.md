Student Placement Management System

Project Overview

The Student Placement Management System is a Python-based application developed to manage student placement activities efficiently.

Features

- Admin Login
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Add Company
- View Companies
- Delete Company
- Search Company
- Add Placement
- View Placements
- Placement Report
- Dashboard
- Placement Percentage
- Company Report
- Export Report
- Data Persistence using JSON Files

Technologies Used

- Python
- JSON
- VS Code

Project Objective

The objective of this project is to simplify the management of student placement records and generate useful placement reports.

Future Improvements

- SQL Database Integration
- Graphical User Interface (GUI)
- Advanced Analytics Dashboard
- Multi-User Login System
